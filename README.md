# PlayKodi #
a cli based remote for kodi
written in bash

i didnt find a good cli based kodi player so i wrote my own

**Current Arguments**

* -p = play
* -m = mute audio
* -s = stop playing

**Examples**

* `playkodi -p http://download.blender.org/demo/movies/Sintel.2010.1080p.mkv`
