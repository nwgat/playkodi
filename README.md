# PlayKodi #
a cli based remote for kodi. written in bash

i didnt find a good cli based kodi remote so i wrote my own

**Current Arguments**

| Feature       |    Argument   |
| ------------- | ------------- |
| Play url     | -u             | 
| Mute Audio   | -m             |
| Change volume 0-100 | -v 0-100| 
| Stop Playing | -s 1           |


**Examples**

* `playkodi -u`
* `url to play` 
* `http://download.blender.org/demo/movies/Sintel.2010.1080p.mkv`
